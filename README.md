# Avon SDK Demo

A lightweight static demo for Avon SDK experiences.

## Pages

- `homepage.html` - entry page for the demo collection.
- `skin-analysis.html` - skin analysis experience.
- `makeup-vto.html` - makeup virtual try-on experience.
- `hair-color.html` - hair color experience.

## Local Usage

Open `index.html` in a browser, or serve the folder with any static file server.
